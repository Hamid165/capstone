# Dhawuh Bumi - E-Commerce Hidroponik

**Dhawuh Bumi** adalah aplikasi web E-Commerce yang dirancang khusus untuk penjualan sayuran hidroponik segar secara online.

Aplikasi ini memudahkan pelanggan untuk mendapatkan sayuran berkualitas langsung dari kebun dengan sistem pemesanan yang mudah, penghitungan ongkos kirim otomatis, dan pembayaran digital yang aman.
